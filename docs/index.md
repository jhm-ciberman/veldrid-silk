# NeoVeldrid

NeoVeldrid is a low-level, cross-platform graphics library for .NET. It provides a powerful, unified interface to a system's GPU, supporting Direct3D 11, Vulkan, OpenGL, and OpenGL ES backends. Applications built with NeoVeldrid can run on Windows, macOS, and Linux without modification.

NeoVeldrid is a maintained fork of [Veldrid](https://github.com/mellinoe/veldrid), modernized with [Silk.NET](https://github.com/dotnet/Silk.NET) bindings and .NET 10 support.
